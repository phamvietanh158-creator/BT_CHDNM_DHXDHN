Công thức Trạng Thái & Tên Đất TCVN

CÔNG THỨC
Đất dính (IP, IL)
IP = WL − WP
IL = (w − WP) / IP
Đất rời (Dr)
Dr = (emax−e) / (emax−emin) × 100%
Trạng thái đất dínhĐiều kiện IL
Cứng (rắn)IL < 0
Nửa cứng0 ≤ IL < 0.25
Dẻo cứng0.25 ≤ IL < 0.50
Dẻo mềm0.50 ≤ IL < 0.75
Dẻo chảy0.75 ≤ IL ≤ 1.0
ChảyIL > 1.0
