    🗂️ Hướng dẫn tạo thư mục & deploy lên GitHub Pages

    Tất cả file HTML đặt lên GitHub Pages → link public cho SV truy cập, không cần server. Kết quả bài làm → Cloudflare Worker → Google Sheet của GV. Mở/đóng chương → chỉ cần sửa config.js rồi push GitHub.
    1
    Tạo repo GitHub: github.com/new → đặt tên VD: nen-mong-baitap → Public → Create. Vào Settings → Pages → Source: Deploy from branch → main / root. URL kết quả: username.github.io/nen-mong-baitap/
    2
    Cấu trúc thư mục:
    📁 nen-mong-baitap/
    ├── index.html      ← Trang chủ (file này)
    ├── config.js       ← GV sửa để mở/đóng chương
    ├── chuong-2/
    │   └── index.html  ← Bài tập chương 2 ✅
    ├── chuong-5/
    │   └── index.html  ← Bài tập chương 5 ✅
    └── chuong-X/
        └── index.html  ← Thêm dần khi build xong
    3
    Mở/đóng chương: Sửa config.js → đổi active: true/false → git commit -m "Mo chuong 2" → git push. Tất cả SV thấy ngay sau ~1 phút. Không cần đụng vào index.html.
    4
    Cloudflare Worker: Đăng ký dash.cloudflare.com → Workers → Create → dán script nhận POST và forward sang Google Apps Script. Copy Worker URL dán vào Admin Panel → Cài đặt.
    5
    Google Apps Script: Mở Google Sheet → Extensions → Apps Script → dán code ghi dòng: timestamp | tên | MSSV | STT | lớp | chương | điểm | thời gian làm bài. Deploy → Web app → Anyone → Copy URL → dán vào Worker.
    6
    Kiểm tra end-to-end: Điền thông tin SV thử → chọn chương đang mở → làm 1 câu → Nộp bài → mở Google Sheet → thấy dữ liệu ✅. Xem cột thời gian làm bài để biết SV mất bao lâu.
