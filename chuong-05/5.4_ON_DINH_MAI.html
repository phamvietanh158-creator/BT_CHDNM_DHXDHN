Dạng 1 — Mái Rời (c = 0)

Tính hệ số ổn định Fs và góc mái dốc cho phép βmax

DẠNG 1
CT

Công thức — Mái Rời

Lý thuyết
Mái Rời — c = 0
Fs = tgφ / tgβ
βmax ≡ φ khi Fs = 1
tgβmax = tgφ / [Fs] (khi cho Fs yêu cầu)
Nhận xét: Mái rời ổn định khi β ≤ φ, bất kể chiều cao H. Fs không phụ thuộc H hay γ.
BM

Bài mẫu — φ = 30°, β = 20°

Cố định
Dữ liệu
φ = 30° → tgφ = 0.577
β = 20° → tgβ = 0.364
Fs yêu cầu = 1.5
Lời giải
Fs = tg30° / tg20° = 0.577 / 0.364 = 1.59
→ Fs = 1.59 > [Fs] = 1.5 ✓ Ổn định
tgβmax = tg30° / 1.5 = 0.577 / 1.5 = 0.385
βmax = arctan(0.385) = 21.1°
Kết quả: Fs = 1.59 (ổn định) · βmax = 21.1°

Dạng 2 — Mái Dính Thẳng Đứng (β = 90°)

Tính chiều cao giới hạn hmax khi φ = 0°

DẠNG 2
CT

Công thức — Mái Thẳng Đứng (φ = 0°)

Lý thuyết
Chiều cao tới hạn
hmax = 4cu / γ (Fs = 1)
hmax = 4cu / (γ · [Fs]) (có Fs yêu cầu)
Tổng quát (có φ): hmax = (4c/γ) · tg(45°+φ/2) / Fs
Lưu ý: cu là lực dính không thoát nước (undrained). Với sét bão hòa: φ = 0°, c = cu.
BM

Bài mẫu — γ = 17.6 kN/m³, cu = 11 kPa

Cố định
γ = 17.6 kN/m³
cu = 11 kPa
[Fs] = 1.5
hmax(Fs=1) = 4×11/17.6 = 2.50 m
hmax(Fs=1.5) = 4×11/(17.6×1.5) = 1.67 m
Chiều cao giới hạn: 2.50 m (Fs=1) · 1.67 m ([Fs]=1.5)

Dạng 3 — Mái Dính (c, φ) — Mặt Trượt Phẳng

Tính hệ số ổn định K theo mặt trượt giả định

DẠNG 3
CT

Công thức — Mặt Trượt Phẳng

Lý thuyết
Hệ số ổn định K
G = ½ · γ · H² / tgα (trọng lượng khối trượt tam giác)
BA = H / sinα (chiều dài mặt trượt)
K = (G·tgφ·cosα + c·BA) / (G·sinα)
Yêu cầu: K ≥ [K] = 1.2 ÷ 1.5
α là góc nghiêng mặt trượt so với phương ngang. Thường lấy α = β + (10°÷20°) để tìm Kmin.
BM

Bài mẫu — H=5m, β=50°, γ=18, c=12, φ=20°, α=55°

Cố định
H = 5 m · β = 50° · α = 55°
γ = 18 kN/m³ · c = 12 kPa · φ = 20°
[K] = 1.2
tgα = tg55° = 1.428
G = ½×18×25/1.428 = 157.6 kN/m
BA = 5/sin55° = 5/0.819 = 6.10 m
Tử: G·tgφ·cosα + c·BA = 157.6×0.364×0.574 + 12×6.10
= 32.91 + 73.2 = 106.1 kN/m
Mẫu: G·sinα = 157.6×0.819 = 129.1 kN/m
K = 106.1 / 129.1 = 0.82
⚠ K = 0.82 < [K] = 1.2 → Mái dốc KHÔNG ổn định! Cần giảm β hoặc tăng c.
