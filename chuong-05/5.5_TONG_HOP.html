5.2 — Trạng thái tại điểm
sinθmax trực tiếp
sinθmax = √[(σx−σz)²+4τ²] / (σx+σz+2c/tgφ)
So sánh với sinφ → kết luận
sinθmax < sinφ: đàn hồi · = : giới hạn · > : phá hoại
5.3 — SCT Terzaghi, móng băng
pgh bài toán phẳng
pgh = ½Nγbγ + Nqq + Ncc
q = γ·hm  ·  Fs = pgh/ptx
φ=0°: Nγ=0, Nq=1, Nc=5.7 → pgh=5.7cu+q
5.3 — SCT Terzaghi, móng đơn
pgh với hệ số hình dạng α
pgh = α₁½Nγbγ + α₂Nqq + α₃Ncc
Vuông: α=(0.8,1.0,1.3) · Tròn: α=(1.2,1.0,1.3)
Nγ, Nq, Nc tra bảng theo φ — không phụ thuộc hình dạng móng
5.4 — Mái đất rời
Hệ số ổn định & góc giới hạn
Fs = tgφ / tgβ
tgβmax = tgφ / [Fs]
c = 0, chỉ phụ thuộc góc ma sát trong φ và góc mái β
5.4 — Đào thẳng đứng (φ=0)
Chiều cao giới hạn
hmax(Fs=1) = 4cu / γ
hcp = 4cu / (γ·[Fs])
Chỉ áp dụng cho đất sét bão hòa φ=0°, đào thẳng đứng
5.4 — Mái dính (c, φ)
Hệ số K, mặt trượt cụ thể
G = ½γH²/tgα  ·  BA = H/sinα
K = (G·tgφ·cosα + c·BA) / (G·sinα)
α = góc nghiêng mặt trượt giả định · [K] = 1.2~1.5
