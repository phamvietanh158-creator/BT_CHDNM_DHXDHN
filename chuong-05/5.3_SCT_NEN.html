Công thức Terzaghi & Bảng tra N

CÔNG THỨC
Móng băng (bài toán phẳng)
pgh = ½·Nγ·b·γ + Nq·q + Nc·c
q = γtb·hm  (tải đất trên đáy móng)
Móng đơn (hệ số hình dạng α)
pgh = α₁·½Nγbγ + α₂·Nqq + α₃·Ncc
Móng tròn: α₁=1.2, α₂=1.0, α₃=1.3
Móng vuông: α₁=0.8, α₂=1.0, α₃=1.3
Hệ số an toàn
[p] = pgh / Fs  ·  Fs = pgh/ptx
Fs ≥ 2~3 thì nền an toàn
