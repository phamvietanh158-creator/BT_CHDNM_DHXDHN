Lý Thuyết — Cộng Lún Từng Phân Tố & Điều Kiện Tắt Lún

LÝ THUYẾT
Các công thức tính Si 1 phân tố
CT1 — Theo e₀, e₁:
Si = (e₀ᵢ − e₁ᵢ)·hᵢ / (1+e₀ᵢ)
CT2 — Theo a, e₀:
Si = aᵢ·Δσᵢ·hᵢ / (1+e₀ᵢ)
CT3 — Theo mv:
Si = mv,i·Δσᵢ·hᵢ [m]
CT4 — Theo E₀:
Si = β·Δσᵢ·hᵢ / E₀ (β=1−2μ²/(1−μ))
Tổng: S = Σ Sᵢ (đến điểm tắt lún)
Điều kiện tắt lún & quan hệ hệ số
Tắt lún: Δσᵢ ≤ 0.2·σbt,i
σbt = Σγ·h (ứng suất bản thân)
Quan hệ a ↔ mv:
mv = a / (1+e₀)
a = mv·(1+e₀)
Tính ngược:
a = S·(1+e₀) / (Δσ·h)
mv = S / (Δσ·h)
Δσ = S·(1+e₀) / (a·h) hoặc S/(mv·h)
📌 Lưu ý đơn vị: S tính ra m → nhân ×100 để ra cm. Hệ số a có đơn vị m²/kN. mv có đơn vị m²/kN. Δσ đơn vị kN/m². h đơn vị m.
Câu hỏi ôn lý thuyết
Q1Điều kiện để dừng cộng lún tại phân tố thứ i là gì?
AΔσᵢ ≤ 0.2·σbt,i
Bσbt,i ≤ 0.2·Δσᵢ
CSᵢ ≤ 0.01 cm
Dz ≥ 5m (độ sâu cố định)
Q2Quan hệ giữa hệ số nén lún a và hệ số nén thể tích mv là gì?
Aa = mv · e₀
Bmv = a / (1+e₀)
Ca = mv / (1+e₀)
Da và mv độc lập nhau
Q3Tính S theo CT3 (mv): Sᵢ = mv·Δσ·h, với mv=0.0002 m²/kN, Δσ=100 kN/m², h=0.5m. S = ?
AS = 0.01 cm
BS = 0.1 cm
CS = 1.0 cm
DS = 10 cm

Nhóm A — Hướng dẫn đầy đủ · A1 – A12

Có gợi ý từng bước · Số liệu sinh theo STT

NHÓM A
A1 — Tính Si theo e₀, e₁
⏳
Gợi ý
Si = (e₀ − e₁) · h / (1 + e₀) [m] → ×100 → cm
A2 — Tính Si theo a, e₀
⏳
Gợi ý
Si = a · Δσ · h / (1 + e₀) [m] → ×100 → cm
Lưu ý: a đơn vị m²/kN
A3 — Tính Si theo mv
⏳
Gợi ý
Si = mv · Δσ · h [m] → ×100 → cm
mv đơn vị m²/kN
A4 — Tính S tổng nền đất (cộng lún nhiều phân tố, cho a & mv)
⏳
Gợi ý
S1 = a₁·Δσ₁·h/(1+e₀₁); S2 = mv2·Δσ₂·h; S = S₁+S₂
Kiểm tra tắt lún: Δσcuối ≤ 0.2·σbt,cuối
A5 — Tính Si theo E₀ (dùng hệ số β)
⏳
Gợi ý
β = 1 − 2μ² / (1−μ) (hệ số nở hông)
Si = β · Δσ · h / E₀ [m] → ×100 → cm
A6 — Tính ngược hệ số nén a (cho S)
⏳
Gợi ý — Đảo CT2
a = S · (1+e₀) / (Δσ · h) (S phải đổi sang m!)
A7 — Tính ngược hệ số nén thể tích mv (cho S)
⏳
Gợi ý — Đảo CT3
mv = S / (Δσ · h) (S phải đổi sang m!)
A8 — Tính ngược Δσ gây lún (cho S, a hoặc mv)
⏳
Gợi ý
Từ CT3: Δσ = S / (mv · h) (S đổi sang m)
A9 — Cộng lún 2 phân tố (cho S từng phân tố)
⏳
Gợi ý
S₁ = a₁·Δσ₁·h/(1+e₀₁); S₂ = a₂·Δσ₂·h/(1+e₀₂)
S = S₁ + S₂
A10 — Cộng lún 3 phân tố + kiểm tra tắt lún
⏳
Gợi ý
Si = mvi·Δσᵢ·h → S = S₁+S₂+S₃
Kiểm tra: Δσ₃ ≤ 0.2·σbt,3?
A11 — Cộng lún 4 phân tố (hỗn hợp a và mv)
⏳
Gợi ý — Dùng CT phù hợp cho từng phân tố
PT1,3 dùng a: S = a·Δσ·h/(1+e₀)
PT2,4 dùng mv: S = mv·Δσ·h
A12 — Xác định chiều sâu ảnh hưởng lún (kiểm tra tắt lún từng điểm)
⏳
Gợi ý
Tại mỗi điểm z: tính σbt=γ·z và Δσ cho trước
So sánh: Δσ ≤ 0.2·σbt? → nếu có → dừng tại đó
Hah = chiều sâu phân tố đầu tiên thỏa điều kiện
