Công thức cộng lún

Lún 1 phân tố
Sᵢ = (e₀ᵢ−e₁ᵢ)·hᵢ / (1+e₀ᵢ)
Hoặc dùng a:
Sᵢ = aᵢ·Δσᵢ·hᵢ / (1+e₀ᵢ)
Hoặc dùng mᵥ:
Sᵢ = mᵥᵢ·Δσᵢ·hᵢ [m] → ×100 [cm]
S = Σ Sᵢ (cộng đến điểm tắt lún)
Điều kiện tắt lún
Δσᵢ ≤ 0.2 · σ_bt,i
σ_bt = γ · z (ứng suất bản thân)
Δσ = k₀ · p_gl (ứng suất tăng thêm)
Nếu điều kiện chưa thoả:
→ tiếp tục chia thêm phân tố

Dạng 1 — Tính Sᵢ Dùng Hệ Số Nén Lún a

Sᵢ = aᵢ·Δσᵢ·hᵢ/(1+e₀ᵢ)

DẠNG 1
BM

Bài mẫu — 3 phân tố h=0.5m

Cố định
Nền 3 phân tố, h=0.5m mỗi cái:
PT1: e₀=0.85, a=0.0003 m²/kN, Δσ=120 kN/m², σ_bt=40 kN/m²
PT2: e₀=0.82, a=0.0002 m²/kN, Δσ=80 kN/m², σ_bt=60 kN/m²
PT3: e₀=0.80, a=0.0002 m²/kN, Δσ=40 kN/m², σ_bt=80 kN/m²
Sᵢ = a·Δσ·h/(1+e₀)
S₁ = 0.0003×120×0.5/(1+0.85) = 0.018/1.85 = 0.00973 m = 0.97 cm
S₂ = 0.0002×80×0.5/(1+0.82) = 0.008/1.82 = 0.0044 m = 0.44 cm
S₃ = 0.0002×40×0.5/(1+0.80) = 0.004/1.80 = 0.0022 m = 0.22 cm
S = 0.97+0.44+0.22 = 1.63 cm
Kiểm tra tắt lún PT3: 0.2×σ_bt = 0.2×80 = 16 kN/m²
Δσ₃ = 40 > 16 → chưa tắt, phải tính thêm PT4!

Dạng 2 — Tính Sᵢ Dùng Hệ Số Nén Thể Tích mᵥ

Sᵢ = mᵥᵢ · Δσᵢ · h

DẠNG 2
BM

Bài mẫu — h=0.4m, 3 phân tố dùng mᵥ

Cố định
Nền 3 phân tố h=0.4m: PT1 mᵥ=0.00015 m²/kN, Δσ=100. PT2 mᵥ=0.00010, Δσ=65. PT3 mᵥ=0.00008, Δσ=35. Tính S₁, S₂, S₃, S tổng.
Sᵢ = mᵥᵢ · Δσᵢ · h
S₁ = 0.00015×100×0.4 = 0.006 m = 0.60 cm
S₂ = 0.00010×65×0.4 = 0.0026 m = 0.26 cm
S₃ = 0.00008×35×0.4 = 0.00112 m = 0.112 cm
S = 0.60+0.26+0.112 = 0.972 cm
