Dạng 1 · Bài mẫu đầy đủ · Không chấm điểm
Đề bài

Cột nhà chịu tải trọng N₀ = 800 kN, M₀ = 0 (đúng tâm). Nền đất đồng nhất 1 lớp: γ = 18.5 kN/m³, c = 18 kPa, φ = 20°. Chiều sâu đặt móng hm = 1.5m, γtb = 20 kN/m³, Fs = 2.5.

Kiểm tra với 2 kích thước:

Phương án 1: B₁ = L₁ = 1.8m (vuông)
Phương án 2: B₂ = L₂ = 2.5m (vuông)
(a) Kiểm tra SCT với từng kích thước  ·  (b) Nhận xét khi B tăng  ·  (c) Chọn kích thước đảm bảo cả kỹ thuật và kinh tế
1Tính SCT nền Rn
Tra bảng Terzaghi với φ=20°: Nc=14.8, Nq=6.4, Nγ=5.4
Móng vuông: α1=0.40, α2=1.00, α3=1.30
Áp lực tầng phủ: q = γ·hm = 18.5×1.5 = 27.75 kPa

PA1 (B=1.8m):
Rn = (1/2.5)·[0.40×5.4×1.8×18.5 + 1.0×6.4×27.75 + 1.3×14.8×18]
= 0.4×[23.9 + 177.6 + 345.8] = 0.4×547.3 = 218.9 kPa

PA2 (B=2.5m):
Rn = 0.4×[0.40×5.4×2.5×18.5 + 177.6 + 345.8]
= 0.4×[33.2 + 177.6 + 345.8] = 0.4×556.6 = 222.6 kPa
(Rn tăng nhẹ khi B tăng do thành phần Nγ·B)
2Tính áp lực đáy móng ptc
Tải đúng tâm M₀=0 → chỉ có ptc đều:
ptc = N₀/(B²) + γtb·hm

PA1: ptc = 800/1.8² + 20×1.5 = 246.9 + 30 = 276.9 kPa
PA2: ptc = 800/2.5² + 20×1.5 = 128.0 + 30 = 158.0 kPa

4Nhận xét và kết luận
Khi B tăng từ 1.8m → 2.5m:
• ptc giảm từ 277 kPa → 158 kPa (giảm 43%) — do diện tích B² tăng
• Rn tăng nhẹ từ 219 kPa → 223 kPa — do thành phần Nγ·B tăng
→ PA1 quá nhỏ (p > Rn); PA2 quá lớn (p chỉ bằng 71% Rn, lãng phí vật liệu)
✅ Kết luận: Chọn B tối ưu ≈ 2.0–2.1m để thỏa ptc ≤ Rn (kỹ thuật) và ptc ≥ 0.7·Rn (kinh tế).
Trong thực tế làm tròn lên B=2.0m, kiểm tra lại: ptc=800/4+30=230 kPa ≤ Rn≈221 kPa — sát giới hạn, chọn B=2.1m là hợp lý.

Móng băng dưới tường chịu tải N₀=180 kN/m, M₀=30 kN·m/m. Nền đất: γ=18.0 kN/m³, c=12 kPa, φ=15°. hm=1.5m, γtb=20 kN/m³, Fs=2.5.

PA1: B₁ = 1.2m
PA2: B₂ = 2.0m
(a) Kiểm tra pmax, pmin từng PA  ·  (b) Nhận xét  ·  (c) Chọn B hợp lý
1Tính Rn — Móng băng (α₁=α₂=α₃=1.0)
φ=15°: Nc=11.0, Nq=3.9, Nγ=2.5. Băng: α₁=α₂=α₃=1.0. q=18.0×1.5=27.0 kPa
PA1 (B=1.2m): Rn=0.4·[1.0×2.5×1.2×18+1.0×3.9×27+1.0×11.0×12]=0.4×[54+105.3+132]=116.5 kPa
Chú ý: móng băng Fs=2.5 → Rn=(1/2.5)×… Tính lại:
Rn=(1/2.5)×[54+105.3+132]=116.5 kPa (thực ra ≈190 kPa với q lớn hơn — xem tính đúng dưới)
Tính đầy đủ PA1:
Rn=(1/2.5)×[1.0×2.5×1.2×18+1.0×3.9×27+1.0×11.0×12] = (1/2.5)×[54+105.3+132] = 116.5 kPa
PA2 (B=2.0m): Rn=(1/2.5)×[2.5×2.0×18+3.9×27+11.0×12]=(1/2.5)×[90+105.3+132]=130.9 kPa
2Tính pmax, pmin
Móng băng: pmax,min = N₀/B + γtb·hm ± 6·M₀/B²

PA1 (B=1.2m):
ptc = 180/1.2 + 20×1.5 = 150 + 30 = 180 kPa
pmax = 180 + 6×30/1.2² = 180 + 125 = 305 kPa
pmin = 180 − 125 = 55 kPa ≥ 0 ✅

PA2 (B=2.0m):
ptc = 180/2.0 + 30 = 90 + 30 = 120 kPa
pmax = 120 + 6×30/4.0 = 120 + 45 = 165 kPa
pmin = 120 − 45 = 75 kPa ≥ 0 ✅

18.0 kN/m³ (dùng γtb theo phần đất trên đáy móng để tính ptc)
1Xác định γtb và tính Rn theo thông số lớp 2
γtb = (γ₁·h₁ + γ₂·(hm−h₁)) / hm = (17.5×2.0 + 19.0×0.5) / 2.5 = 18.0 kN/m³
Thông số tính Rn: dùng lớp 2 (đất dưới đáy móng): c=22 kPa, φ=22°, γ=19.0 kN/m³
φ=22°: Nc=16.9, Nq=7.8, Nγ=7.1. Móng vuông: α₁=0.4, α₂=1.0, α₃=1.3

2Tính pmax, pmin
ptc=N₀/(B²)+γtb·hm  ·  pmax,min=ptc±6·M₀/(B²·L) với L=B (vuông)→ ±6M₀/B³

PA1 (B=2.0m): ptc=1200/4+18×2.5=300+45=345 kPa
pmax=345+6×80/8=345+60=405 kPa ·  pmin=345−60=285 kPa

PA2 (B=2.8m): ptc=1200/7.84+45=153+45=198 kPa
pmax=198+6×80/21.95=198+21.9=220 kPa ·  pmin=198−21.9=176 kPa
